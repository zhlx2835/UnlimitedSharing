GitHub520
=========

一、介绍
----

对 GitHub 说"爱"太难了：访问慢、图片加载不出来。

_注：_ 本项目还处于测试阶段，仅在本机测试通过，如有问题欢迎提 [issues](https://github.com/521xueweihan/GitHub520/issues/new)

本项目无需安装任何程序，通过修改本地 hosts 文件，试图解决：

*   GitHub 访问速度慢的问题
    
*   GitHub 项目中的图片显示不出的问题
    

花 5 分钟时间，让你"爱"上 GitHub。
